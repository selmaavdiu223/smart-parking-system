# Smart Parking Management System

Ky projekt është një sistem për menaxhimin e parkingut inteligjent.

## Funksionet
- Regjistrimi i veturave
- Koha e hyrjes dhe daljes
- Llogaritja automatike e pagesës
- Statusi i vendeve të lira

## Teknologjitë
- C#
- SQL Server
