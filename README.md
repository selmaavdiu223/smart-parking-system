# Smart Parking Management System

## 📌 Përshkrimi
Ky është një sistem për menaxhimin e parkingut që lejon:
- Regjistrimin e hyrjes dhe daljes së veturave
- Llogaritjen automatike të pagesës
- Monitorimin e vendeve të lira dhe të zëna

## 👥 Rolet
- Admin
- Staff

## ⚙️ Teknologjitë
- ASP.NET Core Web API
- PostgreSQL
- HTML, CSS, JavaScript

## 🎯 Qëllimi
Ky projekt po zhvillohet gjatë një semestri për të mësuar full-stack development.
